## 🗂️ Task Details

### Task 1 — Mobile App Sign-Up Flow
**File:** `task1-signup.html`

A personalized multi-step sign-up flow designed for a mobile app experience.

**Features:**
- 4-step onboarding with a live progress indicator
- Interest selection using interactive chip-based UI
- Smooth step transitions with fade animation
- Mobile-first, clean layout with accessible inputs
- Personalized welcome message on completion

**Design Highlights:** Purple gradient theme, rounded card UI, minimal and distraction-free flow.

---

### Task 2 — Promotional Email Template
**File:** `task2-email.html`

A luxury promotional email designed for **DivLuxe Maison**, a fine jewellery brand.

**Features:**
- Desktop/Mobile preview toggle built into the page
- Dark luxury theme with gold (#FCD34D) accents
- Featured product cards with original and sale pricing
- Promo code section and CTA buttons
- Realistic email client chrome (From, Subject fields)

**Design Highlights:** Cormorant Garamond serif typography, radial gradient hero, star particle background.

### Task 4 — E-Commerce Website
**File:** `task4-ecommerce.html`

A full e-commerce storefront UI for **DivLuxe** — a luxury jewellery and premium beauty brand.

**Features:**
- Product listing with category filtering
- Search bar functionality
- Product cards with pricing and add-to-cart buttons
- Elegant black and gold luxury theme
- Fully responsive layout with sticky navigation

**Design Highlights:** Black (#000) base, Cormorant Garamond display type, gold accent throughout, premium brand feel.
